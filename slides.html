2 — Feedback Taxonomy

What are we optimizing toward?

Human AI / LLM Programmatic
Demonstrations SFT Distillation
Preferences RLHF / DPO RLAIF / Constitutional AI
Scalar Reward RM (learned) Verifiable ✓
Binary Outcome Math / Code tests

↑ Richer signal ≠ better. Verifiable > learned when available.
